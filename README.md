# AI-Powered IIS Log Threat Detection & Network Traffic Analysis Dashboard

## Overview

This project was developed during my internship at HTC Global Services.

The dashboard analyzes IIS web server logs in real time, detects suspicious activities using Machine Learning, calculates risk scores, and forecasts future traffic patterns.

## Features

- Real-time IIS Log Monitoring
- Threat Detection using Isolation Forest
- Risk Scoring Engine
- Rule-Based Attack Detection
- Traffic Forecasting using Prophet
- Interactive Streamlit Dashboard
- Live Threat Analytics
- Log Search and Filtering

## Technologies

- Python
- Streamlit
- Pandas
- Scikit-Learn

- IIS Logs
